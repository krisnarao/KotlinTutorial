# VaultGuardian AI

## Intelligent Offline Break Glass Platform

VaultGuardian AI is an AI-powered emergency privileged access platform designed for enterprise environments where traditional secret management systems (such as HashiCorp Vault or CyberArk) become unavailable during outages, cyber attacks, or disaster recovery scenarios.

The platform enables secure emergency access to production secrets while ensuring AI-driven risk assessment, approval workflow, auditing, and governance.

---

# Problem Statement

Modern enterprises rely heavily on centralized secret management platforms.

During planned maintenance, production outages, ransomware attacks, or disaster recovery scenarios, these platforms may become unavailable.

As a result:

- Production engineers cannot access critical credentials.
- Incident resolution time increases.
- Manual break-glass processes are slow.
- Lack of centralized auditing.
- High risk of unauthorized privileged access.

---

# Solution

VaultGuardian AI provides an intelligent offline emergency access platform.

The platform allows users to:

- Register production servers
- Manage application secrets
- Submit emergency Break Glass requests
- Perform AI-based risk assessment
- Route requests for approval
- Provide explainable AI recommendations
- Maintain complete audit history

# Backend Setup

## Prerequisites

- Java 21
- Maven 3.9+
- IntelliJ IDEA (Recommended)

## Clone Repository

```bash
git clone <repository-url>
```

## Navigate

```bash
cd backend
```

## Build

```bash
mvn clean install
```

## Run

```bash
mvn spring-boot:run
```

Backend will start at

```
http://localhost:8080
```
